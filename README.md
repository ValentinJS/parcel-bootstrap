# Parcel base project

Quickly bootstrap a Webapp project using the Parcel bundler

The project comes setup with React, Typescript, Eslint, and Prettier

## Install

Clone the project then
yarn install

yarn start

yarn ci (lint, ts, and build)

yarn build
